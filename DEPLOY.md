## Переменные окружения для продакшена

Создайте эти переменные на вашем хостинге:

```env
DISCORD_CLIENT_ID=ваш_client_id
DISCORD_CLIENT_SECRET=ваш_client_secret
NEXTAUTH_URL=https://ваш-домен.com
NEXTAUTH_SECRET=ваш_секретный_ключ
DISCORD_GUILD_ID=ваш_guild_id
```

## Рекомендуемые платформы для деплоя

1. **Vercel** (рекомендуется)
   - Автоматический деплой из Git
   - Встроенная поддержка Next.js
   - Бесплатный план

2. **Netlify**
   - Простой деплой
   - Хорошая производительность

3. **Railway**
   - Простая настройка
   - Поддержка баз данных

## Команды для деплоя

```bash
# Сборка для продакшена
npm run build

# Запуск продакшен сервера
npm start
```
